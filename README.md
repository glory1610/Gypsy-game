# "Гра в цигана"
Колоду перемішують, всі карти роздають порівну гравцям, гравці бачать свої карти. Мета гри – скинути всі карти. Ходять по черзі, викладаючи карти на стіл сорочкою догори. Перший, хто ходить, оголошує масть і кількість карт, які він кладе, наприклад: «дві черви». Наступні гравці зобов’язані викласти одну або більше карт тієї ж масті, але можуть схитрувати і, оголосивши «кладу три черви», покласти три карти інших мастей, щоб якнайдовше зберігати черви і залишатися в грі. Будь-хто з суперників гравця може заявити «Не вірю!» і відкрити викладені карти. Якщо викриється обман, то всі наявні на столі карти забирає хитрун, а у протилежному випадку – заявник. Програє той, хто залишиться з картами на руках.
